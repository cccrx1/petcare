# 移动端使用指南

## 概述

AI宠物健康管理系统的移动端界面专为手机用户设计，提供简洁、直观的用户体验。

## 访问移动端

### 方式一：直接访问

在手机浏览器中访问：
```
http://your-domain.com/mobile.html
```

### 方式二：桌面端预览

在桌面浏览器中访问：
```
http://localhost:3000/mobile.html
```

建议使用浏览器的开发者工具（F12）切换到移动设备模式进行预览。

## 界面说明

### 顶部状态栏

- 显示当前时间
- 显示网络状态
- 固定在屏幕顶部

### 底部导航栏

包含5个主要功能入口：

1. **首页** - 查看概览和快捷操作
2. **宠物** - 管理宠物档案
3. **AI助手** - 智能聊天助手（中央突出显示）
4. **健康** - 健康监测和预警
5. **我的** - 个人设置

## 主要功能

### 1. 首页

**功能概览：**
- 显示欢迎信息和用户头像
- 宠物数量统计
- 健康记录统计
- 快捷操作入口
- 最新预警信息
- 我的宠物列表

**快捷操作：**
- 🐕 宠物档案 - 快速查看和管理宠物
- 💓 健康监测 - 记录健康数据
- 🏥 兽医咨询 - 在线问诊
- 🍖 喂养方案 - 查看喂养建议

### 2. 宠物档案

**功能：**
- 查看所有宠物列表
- 添加新宠物
- 查看宠物详情
- 编辑宠物信息
- 删除宠物

**宠物卡片显示：**
- 宠物头像
- 宠物名称
- 品种信息
- 健康状态标签

### 3. AI助手

**功能：**
- 与宠物健康助手智能体对话
- 咨询健康问题
- 获取喂养建议
- 行为训练指导

**使用方式：**
1. 点击底部导航栏的"AI助手"按钮
2. 在输入框中输入您的问题
3. 点击发送或按回车键
4. 等待AI助手回复

**常见问题示例：**
- "我家狗狗最近不爱吃饭怎么办？"
- "猫咪应该多久洗一次澡？"
- "如何训练小狗不乱叫？"

### 4. 健康监测

**功能：**
- 选择宠物查看健康数据
- 添加健康记录
- 查看健康趋势
- 查看预警信息

**健康数据类型：**
- 体重记录
- 饮食量
- 饮水量
- 运动量
- 其他健康指标

**预警等级：**
- 🟡 低风险 - 建议继续观察
- 🟠 中风险 - 注意饮食和行为
- 🔴 高风险 - 建议尽快就医

### 5. 个人设置

**功能：**
- 查看个人资料
- 编辑个人信息
- 应用设置
- 关于信息
- 使用帮助
- 退出登录

## 使用技巧

### 1. 快捷导航

- 使用底部导航栏快速切换页面
- 点击首页的快捷操作卡片直接进入功能

### 2. 添加到主屏幕

**iOS设备：**
1. 在Safari中打开移动端页面
2. 点击分享按钮（方形带箭头）
3. 选择"添加到主屏幕"

**Android设备：**
1. 在Chrome中打开移动端页面
2. 点击菜单按钮（三个点）
3. 选择"添加到主屏幕"

### 3. 离线使用

- 当前版本需要网络连接
- 计划支持离线缓存功能

### 4. 数据同步

- 移动端与桌面端数据实时同步
- 支持多设备同时登录

## 常见问题

### Q1: 如何切换语言？

A: 当前版本仅支持中文，后续将支持多语言。

### Q2: 如何添加多只宠物？

A: 进入"宠物"页面，点击右上角的"添加"按钮。

### Q3: AI助手如何使用？

A: 点击底部导航栏中央的"AI助手"按钮，输入问题即可。

### Q4: 健康数据如何记录？

A: 进入"健康"页面，选择宠物后点击"记录"按钮。

### Q5: 如何接收预警通知？

A: 系统会自动推送预警信息，可在"健康"页面查看。

## 隐私与安全

### 数据保护

- 所有数据传输使用HTTPS加密
- 用户密码使用bcrypt加密存储
- 敏感信息不会在本地缓存

### 权限说明

- 相机权限：用于上传宠物照片
- 存储权限：用于保存宠物图片
- 通知权限：用于接收预警通知
